# Chapter 01: k-최근접 이웃(k-NN)을 이용한 이진 분류

## 1. 분석 목적
- 생선의 길이(Length)와 무게(Weight) 데이터를 활용하여 도미(Bream)와 빙어(Smelt)를 분류하는 이진 분류(Binary Classification) 모델 구축.
- 머신러닝의 기초인 k-NN 알고리즘의 작동 원리와 하이퍼파라미터 $k$의 영향력 이해.

## 2. 데이터셋 및 전처리
- **데이터 구성**: 도미 35마리, 빙어 14마리 (총 49개 샘플).
- **특성 공학**: 1차원 리스트 형태의 데이터를 Scikit-learn 입력 형식인 2차원 배열(Matrix)로 재구성.

## 3. 통계적 인사이트 (Critical Analysis)
### 3.1 거리 기반 알고리즘의 한계
- k-NN은 유클리드 거리($L2 \text{ Distance}$)를 사용하여 이웃을 결정함.
- **수식**: $d(p, q) = \sqrt{\sum_{i=1}^{n} (p_i - q_i)^2}$
- 시각화 결과, 무게(y축)와 길이(x축)의 수치 단위가 크게 차이남을 확인. 이는 차후 모델이 특정 변수에 편향(Bias)될 위험이 있으며, 표준화($Z$-score)의 필요성을 시사함.

### 3.2 평가 지표의 신뢰성 문제 (Data Leakage)
- 본 실험의 정확도($Score$)는 1.0(100%)으로 산출됨.
- 이는 훈련 세트와 테스트 세트를 분리하지 않고 동일한 데이터로 평가를 진행했기 때문임. 통계적으로 이는 모델의 **일반화 성능(Generalization Performance)**을 전혀 보장하지 못하는 수치임.

### 3.3 하이퍼파라미터 $k$ ($n\_neighbors$) 실험
- $k=49$(전체 샘플 수)로 설정 시, 모델은 데이터의 개별 패턴을 무시하고 다수 클래스인 '도미'로만 편향된 예측을 수행함.
- 이때의 정확도는 0.7143($35/49$)으로 수렴하며, 이는 모델이 과소적합(Underfitting)되었음을 보여주는 지표임.
