# [Chapter 1-3] k-NN을 이용한 생선 분류 기초

### 1. 공부한 내용 요약
- **데이터셋**: 도미 35마리, 빙어 14마리의 길이와 무게 데이터
- **핵심 개념**: k-최근접 이웃(k-NN) 알고리즘. 주변에 가장 가까운 $k$개의 데이터를 확인해 다수결로 종류를 판별함
- **작업 흐름**: `matplotlib` 시각화 -> `zip()`을 활용한 2차원 리스트 생성 -> `sklearn` 모델 학습 및 평가

### 2. 분석 및 궁금한 점 (My Analysis & Questions)
- **정확도 100%에 대한 의문**: 현재 점수가 1.0(100%)이 나왔는데, 이건 학습에 쓴 데이터를 그대로 평가에 썼기 때문임. 실제 연구실에서 처음 보는 데이터를 분류해야 할 때는 이 방식이 통하지 않을 것 같음. 훈련 세트와 테스트 세트를 나누는 법을 확실히 익혀야겠음.
- **데이터 스케일의 차이**: 산점도를 그려보니 무게(y축)의 수치 범위가 길이(x축)보다 훨씬 큼. k-NN은 거리를 계산하는 방식인데, 이렇게 단위가 차이 나면 무게 변수에 모델이 너무 의존하게 되지 않을까 걱정됨. 통계 전공에서 배운 표준화 개념이 여기서 쓰일 것 같음.
- **최적의 $k$값**: 주변 이웃을 몇 개($k$)까지 보느냐에 따라 결과가 달라지는데, $k$가 너무 커지면 데이터의 세밀한 특징을 놓칠 것 같음. 통계적으로 최적의 $k$를 정하는 명확한 기준이 있는지 궁금함.
