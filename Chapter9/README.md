# [Chapter 9-1] 순차 데이터와 순환 신경망 기초

### 1. 주요 개념 및 용어 정리
- **순차 데이터 (Sequential Data)**: 데이터의 요소들이 특정 순서를 가지고 있으며, 그 순서가 어긋날 경우 정보의 의미가 훼손되는 데이터(예: 문장, 주가 데이터 등)를 이해했다.
- **순환 신경망 (RNN)**: 뉴런의 출력이 다시 입력으로 들어가는 순환 고리를 통해 시퀀스 데이터를 처리하는 인공 신경망 구조를 학습했다.
- **셀 (Cell)**: RNN에서 반복적으로 실행되는 핵심 연산 단위를 의미하며, 이전 단계의 정보를 유지하는 역할을 한다.
- **은닉 상태 (Hidden State)**: 셀이 출력하는 값이 다음 단계의 입력으로 다시 전달될 때의 상태값을 의미하며, 이를 통해 시퀀스의 맥락(Context)을 파악한다.
