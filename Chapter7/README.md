# [Chapter 7-1] 인공 신경망 (Artificial Neural Network)

### 1. 공부한 내용 요약
- **데이터셋 준비**: 10가지 패션 아이템으로 구성된 Fashion MNIST 데이터를 로드하고, 픽셀값을 0~1 사이로 정규화했다.
- **로지스틱 회귀 모델 (Baseline)**: `SGDClassifier`를 이용해 기존 머신러닝 방식의 성능을 측정했으며, 이미지 데이터의 특성 수(784개)가 많아질수록 학습 시간이 길어지고 성능 최적화가 까다로움을 확인했다.
- **인공 신경망 설계**: 케라스의 `Dense` 클래스를 사용하여 10개의 뉴런을 가진 출력층을 생성했으며, 다중 분류를 위한 소프트맥스 활성화 함수를 지정했다.
- **모델 설정 및 훈련**: `sparse_categorical_crossentropy` 손실 함수와 `accuracy` 지표를 설정하여 모델을 컴파일하고, 5 에포크 동안 학습을 진행했다.



### 2. 분석 및 비판적 검토 (My Analysis & Questions)
- **로지스틱 회귀와 신경망의 수리적 연관성**: 이번 실습에서 만든 단층 신경망은 사실상 4장에서 배운 다항 로지스틱 회귀와 수리적으로 동일하다. 다만 이를 케라스라는 프레임워크를 통해 추상화함으로써, 향후 층(Layer)을 더 깊게 쌓을 수 있는 확장성을 확보했다는 점에 의미가 있다.
- **성능의 정체 현상**: 훈련 세트에 대한 정확도는 약 85% 수준에서 머물렀다. 이는 현재 모델이 은닉층(Hidden Layer)이 없는 단순한 선형 결합 구조이기 때문이며, 데이터의 복잡한 비선형 패턴을 학습하기에는 모델의 용량(Capacity)이 부족함을 시사한다.
- **통계적 손실 함수의 선택**: 타깃값이 정수 형태(레이블 인코딩)인 경우 `sparse_` 접두사를 붙여 원-핫 인코딩 과정 없이 바로 교차 엔트로피를 계산하는 방식의 효율성을 확인했다. 통계학에서 다항 분포의 우도를 최대화하는 것과 손실 함수를 최소화하는 것이 수리적으로 어떻게 맞닿아 있는지 다시금 상기했다.
- **검증 세트 성능의 의미**: `evaluate()` 메서드를 통해 확인한 검증 세트의 점수가 훈련 세트와 큰 차이가 없음을 확인했다. 이는 아직 모델이 과대적합될 만큼 충분히 복잡하지 않음을 의미하며, 다음 단계에서 층을 추가했을 때 과대적합 문제가 어떻게 대두될지 주의 깊게 살펴볼 필요가 있다.
