# [Chapter 6-1] 군집 알고리즘 (Clustering Algorithm)

### 1. 공부한 내용 요약
- **비지도 학습(Unsupervised Learning)**: 타깃(Label)이 주어지지 않은 상태에서 데이터 사이의 패턴을 찾아내고 유사한 샘플을 묶는 학습 방식을 이해했다.
- **넘파이 배열 처리**: `np.load()`로 3차원 배열 데이터를 불러오고, `reshape()`를 사용하여 샘플별 픽셀 데이터를 1차원 배열로 평탄화했다.
- **픽셀 평균값 계산**: 사과, 파인애플, 바나나 각 그룹의 픽셀별 평균값을 도출하여 시각화했으며, 이를 통해 각 클래스의 수리적 특징을 파악했다.
- **유사 샘플 추출**: 특정 클래스의 평균값과 개별 데이터 사이의 절대 오차(`np.abs`)를 계산하고, `np.argsort()`를 활용해 오차가 가장 작은 100개의 샘플을 성공적으로 골라냈다.

### 2. 분석 및 비판적 검토 (My Analysis & Questions)
- **수동 군집화의 한계**: 이번 실습에서는 우리가 이미 과일의 종류(1~100번은 사과 등)를 알고 있는 상태에서 평균값을 구했다. 하지만 실제 비지도 학습 환경에서는 타깃을 전혀 알 수 없으므로, 현재의 방식처럼 '정답을 먼저 나누고 평균을 구하는 것'은 불가능하다는 점을 인지했다.
- **거리 척도의 선택**: 유사도를 측정하기 위해 평균값과의 절대 오차(L1 노름과 유사한 개념)를 사용했다. 만약 유클리드 거리(L2 노름)를 사용했다면 결과가 어떻게 달라졌을지, 그리고 이미지 데이터에서 어떤 거리 척도가 더 강건(Robust)한지 통계적으로 분석해 볼 가치가 있다.
- **데이터 구조의 중요성**: 2차원 이미지를 1차원으로 펼치면 픽셀 간의 공간적 정보(상하좌우 인접성)가 손실된다. 그럼에도 불구하고 단순 평균값만으로 높은 분류 성능을 보인다는 사실이 흥미로웠으며, 더 복잡한 이미지에서도 이 방식이 유효할지 의문이 들었다.
- **자동화된 알고리즘의 필요성**: 현재는 수동으로 평균을 내고 비교했지만, 데이터가 수만 개로 늘어나고 클래스 개수를 모를 때는 적용이 어렵다. 다음 장에서 배울 **K-평균(K-Means)** 알고리즘이 어떻게 이 과정을 자동화하고 최적의 중심점(Centroid)을 찾아내는지 심도 있게 학습할 계획이다.

# [Chapter 6-2] K-평균 군집 알고리즘

### 1. 공부한 내용 요약
- **K-Means Clustering**: 타깃 레이블이 없는 상태에서 데이터를 $k$개의 군집으로 묶는 대표적인 비지도 학습 알고리즘을 실습했다.
- **데이터 변환 및 학습**: 3차원 과일 데이터를 2차원으로 평탄화하여 `KMeans` 모델에 입력했으며, `fit()` 메서드를 통해 군집화를 수행했다.
- **중심점 및 레이블 확인**: `labels_` 속성을 통해 각 샘플이 할당된 클러스터를 확인하고, `cluster_centers_`를 통해 각 군집의 수리적 중심점을 추출했다.
- **엘보우 방법(Elbow Method)**: 하이퍼파라미터 $k$를 변화시키며 이너셔(Inertia) 값을 플롯하여, 모델의 복잡도와 효율성이 균형을 이루는 최적의 지점을 탐색했다.

### 2. 분석 및 비판적 검토 (My Analysis & Questions)
- **무작위성의 영향력**: K-평균은 초기 중심점을 무작위로 설정하기 때문에 실행할 때마다 결과가 달라질 수 있는 불안정성을 내포하고 있다. 실무에서는 이를 보완하기 위해 여러 번 초기화를 반복하는 `n_init` 매개변수 설정이 필수적임을 인지했다.
- **이너셔 지표의 통계적 해석**: 이너셔는 클러스터 내 오차 제곱합(SSE)과 동일한 개념이다. $k$가 샘플 개수와 같아지면 이너셔는 0이 되므로, 무조건 낮은 값을 찾기보다 감소 폭이 둔화되는 '엘보우' 지점을 찾는 통계적 판단력이 중요함을 깨달았다.
- **차원의 저주와 거리 척도**: 현재는 10,000차원의 픽셀 데이터를 그대로 사용했으나, 차원이 높아질수록 샘플 간의 거리 차이가 모호해지는 현상이 발생할 수 있다. 유클리드 거리에 기반한 K-평균이 고차원 이미지 데이터에서도 항상 최선의 유사도를 보장하는지에 대해서는 비판적인 검토가 필요해 보인다.
- **중심점 시각화의 유용성**: 각 군집의 중심점을 이미지로 출력했을 때, 해당 군집이 사과, 바나나, 파인애플의 특징을 명확히 반영하고 있음을 확인했다. 이는 비지도 학습 모델이 데이터의 잠재적인 물리적 특성을 성공적으로 학습했음을 보여주는 강력한 증거라고 판단했다.
